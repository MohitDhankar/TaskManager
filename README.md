# Task Manager (React MVP)

A Trello-style task board built with React 18, drag-and-drop, and localStorage persistence.

## Features
- 3-column board: To-Do, In-Progress, Completed  
- Create, edit, delete tasks (modal)  
- Drag & drop across columns  
- Priority, due-date, status filters  
- Sort by newest / oldest / closest due  
- Duplicate title auto-numbering  
- Fully responsive  

## Tech
React 18 | react-beautiful-dnd | dayjs | Tailwind CSS

## Run Locally
git clone https://github.com/MohitDhankar/TaskManager.git
cd TaskManager
npm install
npm start
